Block Breaker

13 August 2020

A remake of the classic Breakout game!

This game was a tutorial in the GameDev.tv Complete C# Unity Developer 2D course.

Additional features I added include the life display, power up system and the ability to dynamically set the edge colliders in the game based on the resolution the game is running at.

Controls:

Move the mouse to move the paddle.
Left click the mouse to release the ball.
