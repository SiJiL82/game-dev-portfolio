Stuck in a While Loop

08 October 2020

My submission for the Ludum Dare 47 game jam – the first program I’ve written outside of tutorials, and my first time using 3D in Unity!

The game is stuck in a never ending while loop. Read the pseudo code and use the manipulator gun to alter the game objects to break out of the loop.

Controls:

Middle click to absorb a shape.
Left click to set the shape to what you’ve absorbed.
Right click to set the colour.
Click the ? in game to see the controls.

I didn’t get as many levels in as I wanted, but pretty happy with how it turned out for my first ever project, and it got some nice reviews from people that tried it.

Eventually I plan to expand the game and turn it into a training tool for people learning to program.
